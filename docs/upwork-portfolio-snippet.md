## Title
Telegram AI Lead Assistant (MVP) — FastAPI + Telegram + Automation Flow

## Description
Built a business-focused automation MVP that processes inbound Telegram messages and turns them into structured leads.

### What it does
- Receives inbound webhook events
- Classifies messages as **lead / faq / support**
- Extracts basic lead fields (name, phone, intent)
- Stores records in CSV for reporting
- Sends real-time manager notifications to Telegram
- Exposes health and stats endpoints

### Stack
Python, FastAPI, Uvicorn, Docker, Telegram Bot API

### Why this matters for clients
- Faster response handling
- Better lead organization
- Clear handoff from chat to sales/manager
- Easy path to CRM integration
